## 1. 概述

过去一周（按本次运行时间回溯 24*7 小时）与 OpenAI 相关的高价值更新，主要集中在三点：第一，API 与 SDK 进入高频迭代，3 月 12-13 日连续出现模型/接口与客户端更新；第二，Codex 与开发流程层面出现可见波动，既有 GitHub 预发布快节奏推进，也有状态页记录的短时可用性事件；第三，安全与公司层面同步推进，包括提示注入防护方法论公开，以及围绕企业安全评测能力的并购动作。整体上，对开发者最直接的影响是：需要关注 SDK 版本与 API 变更同步、评估 Sora 视频接口变化、并为工作流加入状态页监控兜底。

  - 发布时间：2026-03-11（来源页面时区未明）
