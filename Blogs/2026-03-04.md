# 2026-03-04

## 1. 概述

过去 24 小时内（截至 2026-03-04 08:00 CST），可核验的 OpenAI 新动向主要有 3 点：第一，`GPT-5.3 Instant` 已在官方发布页与 ChatGPT Release Notes 同步披露，重点是提升日常对话流畅度、网页检索整合质量与回答准确性；第二，OpenAI 同步发布 `GPT-5.3 Instant System Card`，说明该模型的安全缓解框架延续 GPT-5.2 Instant 体系；第三，Codex 工程面出现连续预发布（`openai/codex` 的 `0.108.0-alpha.1/.2/.3`），并且状态页记录了一起 ChatGPT 文件上传错误事件且已恢复。除这些外，API/SDK 官方渠道未发现新的 24 小时内稳定版发布。

## 2. 官方发布与公告

- **GPT-5.3 Instant: Smoother, more useful everyday conversations**
  - 发布时间：March 3, 2026（来源页仅显示日期，未给出具体时区与时分）
  - 来源：https://openai.com/index/gpt-5-3-instant/
  - 要点：
    - 官方将其定义为 ChatGPT 最常用模型的一次更新，关注“更顺滑、更有用”的日常对话体验。
    - 发布说明强调三类直接变化：更少不必要拒答/免责声明、更好的网页信息整合、更可靠的回答准确性。
    - 这类更新对普通用户通常是“默认生效”，无需手动迁移接口。

- **GPT-5.3 Instant System Card**
  - 发布时间：March 3, 2026（来源页仅显示日期，未给出具体时区与时分）
  - 来源：https://openai.com/index/gpt-5-3-instant-system-card/
  - 要点：
    - 系统卡明确 GPT-5.3 Instant 的安全缓解方法总体沿用 GPT-5.2 Instant 方案。
    - 对企业与开发团队的意义是：可将该文档作为合规、风险评审和模型选型的官方依据之一。
    - 页面提供了系统卡原文入口，便于进行条款级复核。
